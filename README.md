# Typeshi

Typeshi is a very basic command-line task manager I put together in Python. 
I built it because I just wanted a fast, straightforward way to keep track of my daily tasks from the terminal without dealing with complicated features or heavy software.

The code is kept super simple and easy to understand, making it great for beginners or anyone who just wants a raw, functional to-do list.

## Features

- Add new tasks to your list
- View all your tasks in one place
- Mark tasks as complete 
- Edit tasks if you made a mistake
- Delete specific tasks
- Clear the entire list at once
- Saves your tasks to a text file automatically so you don't lose them

## How to use

Just run the script and it will ask you what you want to do:

```bash
python main.py
```

Then just type a command like add, list, complete, edit, delete, clear, or exit.
